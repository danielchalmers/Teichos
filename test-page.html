How to Use

  1. Right-click the Teichos extension icon and select "Options"
  2. Add a new group or use the default 24/7 group
  3. Add filters with regex patterns (e.g., .*facebook\.com.*)
  4. Try navigating to a matching URL - it will be blocked!

Example Patterns to Try

Add these patterns in the options page to test blocking:

  • .*example\.com.* - Blocks any URL containing example.com
  • .*facebook\.com.* - Blocks Facebook
  • .*youtube\.com/watch.* - Blocks YouTube videos
  • .*(facebook|twitter|instagram)\.com.* - Blocks multiple social media sites
